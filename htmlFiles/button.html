Buttons

Button indicates the user action or event, such as submitting form, add some product to cart, openning a dialogue box, cancelling an action or deleting something so for all this action or event we need simply a button.

Types of Buttons

IQRA UI includes diffrent buttons like primary button, link button,icon button, outline button. Every button has their own meaning and color. For using Iqra button components, you just need to add btn class and whatever button you have to liked add the class of it in your HTML code.

Different color's button

For using the IQRA UI's button component just add btn class to element. Add btn-color class according to your need. Like in submit form we need two buttons save and cancle. so add btn-primary class to save button and add btn-secondary class to cancel button.

Link buttons

For using the IQRA UI's link button component just add link-btn class to element and if you want that your button have round border then add bprder-round class to it.

Icon Buttons

You can create icon button by just adding icon to button and add whatever class you want to add like btn-primary, btn-seconadry etc.

Floating Buttons

We always see the floating button with plus sign in front of our email list.If you've been using an Android , there's a good chance that you have seen this button on some of the apps. For using the floating button just add the class floating-btn to element.
