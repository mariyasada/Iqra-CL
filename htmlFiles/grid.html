Grid

A grid is a collection of horizontal and vertical lines creating a pattern against which we can line up our design elements. A grid can be used to organize graphic elements in relation to a page.

Type of Grid

In this grid, the Item and description of the item take an equal part of the grid. You can use this by adding class grid-50-50-container to element.
