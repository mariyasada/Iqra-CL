Card

A card is any type of custom element used for easy showing of contents that comprises various types of objects like title, image, button, etc.

Types of cards

IQRA UI includes card with badges, card with dismiss button, text only cards etc. In code of card, e-commerce card is same as first card , but image and span are diffrent, discount and price and padding 1rem are added. If you want to use this card then add card class name to element. suppose you want to use card for your e-commerce website then add card-for-ecommerce class to element.
