Avatars

Avatar componet is typically used to display image like circular profile picture or company logos.This component will allow you to change the size, outline and add radius dynamically. We use text,image with outline and image avatars in many places.

Types of Avatar

In IQRA UI, Avatar are available in diffrent sizes like avatar-xl, avatar-lg, avatar-md, avatar-sm, avatar-xsm. For using IQRA UI's avatar component, you just need to add the class avatar with their size like avatar-xl in your html file. you can see the code below.
